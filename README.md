A comprehensive, user-centric **web-based health & fitness tracking application** developed as part of an MCA university Agile project. The system helps users monitor and improve their overall wellness through four integrated modules: **Fitness**, **Diet**, **Progress**, and **Profile**.

Built with a focus on simplicity, responsiveness, and real-time user interaction using modern web technologies and Agile methodology.
